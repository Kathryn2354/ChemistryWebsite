This is a spectrometer project by Dr. Sherrer.

This project is collaborated with student taking computer science classes. We have create a website for people to upload the results.

If you want to get to the website, open any web application first before you type in cdspecollab.smcm.edu.

This is how to use the cdspeccollab.smcm.edu website:

If you want to upload a CVS file to that website, please click on the upload botton.

This will lead you to a create page, which will ask you to type in a title for you CVS file, what it is run by, you must type in a description for your file and the chractor limit: 2000, protein concentration, curvette Pathlength, number of amino acids, you can choose make the CVS file visible to Students, or Visible to public. You need to click on choose file button in create page to add the CVS file.

Once you fill in all of the required info in the create page, you can then press summit button.

The website will create a place in the table view in the CD Spectroscopic Collaboration page.

If you want to multigraph a graph, you must select the graph that match it data info first before you click on the multigraph button.

If you want to create an account on the website click the Sign Up page in navagation bar.

You must enter you email first, then you must enter you own created password this account in sign up page after you finish filling al of the required info you can click on the sign up summit button.

You must go in your email and click on the varification link in order to verify your account.

If you want login with your varify account, you must click the Sign In page in the navagation bar.

Your login username will be the begin part of the email without email ending part.

You need to enter the verifyed password you created for your account on the website.

You can make the choice of allowing the website to remember your account if you like, this part is optional.

if you forget your account password for this website click on the forgot password? button at the bottom of the Sign In page. This will take to a password reset page where you can reset your account password.

Once you enter the password reset page, you need to enter the email address and press summit button that can reset you password.

You need to go to your email and click on the reset password link in order to change your account password.

There is an search bar in CD Spectroscopic Collaboration page.

You can go on the right side and on the top of the CD Spectroscopic Collaboration page on the table view and type that exact file name you have given it during the create page.

There is an previous button and next button for the table view at the bottom, so you see how many people put entries with CVS file.
